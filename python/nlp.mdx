## Core NLP toolkits

<Columns cols={2}>
  <Card title="spaCy" icon="language" href="https://github.com/explosion/spaCy">
    Industrial-strength NLP with Python and Cython. Fast tokenization, POS tagging, NER, dependency parsing, and word vectors. Production-ready.
  </Card>
  <Card title="NLTK" icon="book" href="https://www.nltk.org/">
    Leading platform for building Python programs that work with human language data. Includes corpora, tokenizers, stemmers, parsers, and classifiers.
  </Card>
  <Card title="TextBlob" icon="comment" href="http://textblob.readthedocs.io/en/dev/">
    Consistent API for common NLP tasks including sentiment analysis, noun phrase extraction, translation, and spelling correction. Built on NLTK and Pattern.
  </Card>
  <Card title="Polyglot" icon="globe" href="https://github.com/aboSamoor/polyglot">
    Multilingual text (NLP) processing toolkit. Supports tokenization, NER, POS tagging, and language detection across 130+ languages.
  </Card>
</Columns>

## Transformers and LLMs

<Columns cols={2}>
  <Card title="Transformers (Hugging Face)" icon="robot" href="https://github.com/huggingface/transformers">
    Deep learning library with thousands of pre-trained models for NLP (and vision/audio). The standard library for anything related to Large Language Models — BERT, GPT, T5, LLaMA, and more.
  </Card>
  <Card title="Haystack" icon="magnifying-glass" href="https://github.com/deepset-ai/haystack">
    Framework for building industrial-strength applications with Transformer models and LLMs. Powers RAG pipelines, QA systems, and document search.
  </Card>
</Columns>

<Card title="CometLLM" icon="chart-bar" href="https://github.com/comet-ml/comet-llm">
  Track, log, visualize, and evaluate your LLM prompts and prompt chains across experiments.
</Card>

## Dialogue and intent systems

<Columns cols={2}>
  <Card title="Rasa" icon="comments" href="https://github.com/RasaHQ/rasa">
    Machine learning framework to automate text- and voice-based conversations. Used for building contextual AI assistants and chatbots.
  </Card>
  <Card title="Snips NLU" icon="brain" href="https://github.com/snipsco/snips-nlu">
    Natural Language Understanding library for intent classification and entity extraction. Runs fully on-device without cloud dependencies.
  </Card>
</Columns>

<Card title="DrQA" icon="question" href="https://github.com/facebookresearch/DrQA">
  Facebook Research's system for reading Wikipedia to answer open-domain questions. Combines a document retriever with a machine reading comprehension model.
</Card>

## Topic modelling

<AccordionGroup>
  <Accordion title="Topic modelling libraries" icon="diagram-project">
    | Library | Description |
    |---|---|
    | [Gensim](https://github.com/RaRe-Technologies/gensim) | Topic modelling for humans. Implements LDA, LSI, word2vec, and doc2vec efficiently for large corpora. |
    | [BigARTM](https://github.com/bigartm/bigartm) | Fast topic modelling platform based on Additive Regularization of Topic Models (ARTM). Scales to billions of tokens. |
    | [tweetopic](https://centre-for-humanities-computing.github.io/tweetopic/) | Blazing fast short-text topic modelling for Python. Designed for social media text. |
    | [topicwizard](https://github.com/x-tabdeveloping/topic-wizard) | Interactive topic model visualization and interpretation framework. |
  </Accordion>
</AccordionGroup>

## String matching and distance

<Columns cols={2}>
  <Card title="FuzzyWuzzy" icon="spell-check" href="https://github.com/seatgeek/fuzzywuzzy">
    Fuzzy string matching in Python using Levenshtein distance. Useful for deduplication, record linkage, and search.
  </Card>
  <Card title="Neofuzz" icon="bolt" href="https://github.com/x-tabdeveloping/neofuzz">
    Blazing fast, lightweight, and customizable fuzzy and semantic text search. FuzzyWuzzy/thefuzz compatible API.
  </Card>
  <Card title="jellyfish" icon="fish" href="https://github.com/jamesturk/jellyfish">
    Python library for approximate and phonetic matching of strings. Implements Jaro-Winkler, NYSIIS, Metaphone, and more.
  </Card>
  <Card title="editdistance" icon="ruler" href="https://pypi.org/project/editdistance/">
    Fast implementation of edit distance (Levenshtein distance) in Python. C extension for performance.
  </Card>
</Columns>
