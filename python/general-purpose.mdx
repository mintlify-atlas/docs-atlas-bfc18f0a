## Core frameworks

<Columns cols={2}>
  <Card title="scikit-learn" icon="python" href="https://scikit-learn.org/">
    The standard Python ML toolkit built on top of SciPy. Covers classification, regression, clustering, dimensionality reduction, model selection, and preprocessing with a consistent API.
  </Card>
  <Card title="XGBoost" icon="bolt" href="https://github.com/dmlc/xgboost">
    Parallelized optimized general-purpose gradient boosting library. Widely used in tabular ML competitions and production systems.
  </Card>
  <Card title="CatBoost" icon="bolt" href="https://github.com/catboost/catboost">
    Gradient boosting on decision trees with native categorical feature support. Easy to install; supports CPU and multi-GPU computation.
  </Card>
  <Card title="LightGBM" icon="bolt" href="https://github.com/Microsoft/LightGBM">
    Microsoft's fast, distributed, high-performance GBDT framework. Excels at ranking, classification, and other tasks on large datasets.
  </Card>
</Columns>

## Bayesian inference and probabilistic programming

<Columns cols={2}>
  <Card title="PyMC" icon="chart-line" href="https://github.com/pymc-devs/pymc">
    Markov Chain Monte Carlo (MCMC) sampling toolkit for Bayesian statistical modelling.
  </Card>
  <Card title="pomegranate" icon="diagram-project" href="https://github.com/jmschrei/pomegranate">
    Hidden Markov Models and probabilistic graphical models for Python, implemented in Cython for speed.
  </Card>
  <Card title="pgmpy" icon="diagram-project" href="https://github.com/pgmpy/pgmpy">
    Python library for working with Probabilistic Graphical Models — structure learning, parameter estimation, and inference.

  <Accordion title="Ensemble methods" icon="layer-group">
    | Library | Description |
    |---|---|

## Time series

<Columns cols={2}>
  <Card title="sktime" icon="clock" href="https://github.com/alan-turing-institute/sktime">
    Unified framework for machine learning with time series — classification, regression, clustering, and forecasting with a scikit-learn compatible API.
  </Card>
  <Card title="skforecast" icon="chart-line" href="https://github.com/skforecast/skforecast">
    Time series forecasting using any scikit-learn-compatible regressor, including LightGBM, XGBoost, CatBoost, and Keras.
  </Card>
  <Card title="prophet" icon="calendar" href="https://facebook.github.io/prophet/">
    Fast automated time series forecasting framework by Facebook. Handles seasonality, holidays, and trend changes with minimal configuration.
  </Card>
  <Card title="dtaidistance" icon="ruler" href="https://github.com/wannesm/dtaidistance">
    High-performance library for time series distances (DTW) and time series clustering.
  </Card>
</Columns>

## Hyperparameter optimization

<Columns cols={2}>
  <Card title="Optuna" icon="sliders" href="https://github.com/optuna/optuna">
    Automatic hyperparameter optimization framework with a clean API, pruning, and parallelism support. Particularly designed for machine learning.
  </Card>
  <Card title="bayeso" icon="brain" href="https://github.com/jungtaekkim/bayeso">
    Simple but essential Bayesian optimization package with support for common kernels and acquisition functions.
  </Card>
  <Card title="Opytimizer" icon="magnifying-glass" href="https://github.com/gugarosa/opytimizer">
    Python-based meta-heuristic optimization techniques including swarm intelligence and evolutionary algorithms.
  </Card>
  <Card title="Optunity" icon="sliders" href="https://optunity.readthedocs.io/en/latest/">
    Library for automated hyperparameter optimization; drop-in replacement for grid search with a lightweight API.
  </Card>
</Columns>

## Model monitoring and data quality

<Columns cols={2}>
  <Card title="Evidently" icon="chart-bar" href="https://github.com/evidentlyai/evidently">
    Interactive reports to analyze ML models during validation or production monitoring. Detects data drift, model degradation, and data quality issues.
  </Card>
  <Card title="Deepchecks" icon="shield-check" href="https://github.com/deepchecks/deepchecks">
    Validation and testing of ML models and data during development, deployment, and production — covering performance, integrity, and distribution.
  </Card>
  <Card title="NannyML" icon="bell" href="https://bit.ly/nannyml-github-machinelearning">
    Python library for estimating post-deployment model performance without access to ground-truth targets. Captures data drift impact on performance.
  </Card>
  <Card title="cleanlab" icon="broom" href="https://github.com/cleanlab/cleanlab">
    Standard data-centric AI package for data quality and ML with messy, real-world labels. Finds and fixes label errors automatically.
  </Card>
</Columns>

## Miscellaneous utilities

<AccordionGroup>
  <Accordion title="Explainability and interpretability" icon="magnifying-glass">
    | Library | Description |
    |---|---|
    | [InterpretML](https://github.com/interpretml/interpret) | Implements Explainable Boosting Machine (EBM) and visualization tools for glass-box and black-box models. |
    | [Shapash](https://github.com/MAIF/shapash) | Visualization library displaying explicit labels for model explanations that non-experts can understand. |
    | [lime](https://github.com/marcotcr/lime) | Model-agnostic explanation of black-box classifiers. Works for any model with two or more classes. |
  </Accordion>

    |---|---|
    | [Surprise](https://surpriselib.com) | Scikit-inspired library for building and analyzing recommender systems. |

    | [LightFM](https://making.lyst.com/lightfm/docs/home.html) | Popular recommendation algorithms for both implicit and explicit feedback. |
    | [Cornac](https://github.com/PreferredAI/cornac) | Comparative framework for multimodal recommender systems leveraging auxiliary data. |
  </Accordion>
  <Accordion title="Online and stream learning" icon="water">
    | Library | Description |
    |---|---|
    | [River](https://github.com/online-ml/river) | Framework for general purpose online machine learning, where models learn one sample at a time. |
    | [scikit-multiflow](https://github.com/scikit-multiflow/scikit-multiflow) | ML framework for multi-output/multi-label and stream data. |
  </Accordion>
  <Accordion title="Outlier detection" icon="triangle-exclamation">
    | Library | Description |
    |---|---|
    | [PyOD](https://github.com/yzhao062/pyod) | Comprehensive Python toolkit for detecting outlying objects in multivariate data, including neural network-based methods. |
