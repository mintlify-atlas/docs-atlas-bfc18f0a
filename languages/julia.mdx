Julia combines Python-level expressiveness with C-level performance, making it a natural fit for computationally intensive ML workloads. Its ecosystem has grown substantially since 2018, with mature frameworks for deep learning, probabilistic programming, and large-scale data manipulation.

<Info>
  Julia packages often have Python interop via [PythonCall.jl](https://github.com/cjdoris/PythonCall.jl). For Python ML tools, see the [Python general-purpose ML](/python/general-purpose) page.
</Info>

## General-purpose ML

<Columns cols={2}>
  <Card title="MLJ.jl" icon="robot" href="https://github.com/alan-turing-institute/MLJ.jl">
    A Julia machine learning framework from the Alan Turing Institute — composable, model-agnostic, with a scikit-learn-style API.
  </Card>
  <Card title="Flux.jl" icon="bolt" href="https://fluxml.ai/">
    The primary Julia deep learning library. Relax! Flux is the ML library that doesn't make you tensor.
  </Card>
  <Card title="Knet.jl" icon="graduation-cap" href="https://github.com/denizyuret/Knet.jl">
    Koç University deep learning framework with a define-by-run approach, supporting automatic differentiation and GPU acceleration.
  </Card>
  <Card title="ScikitLearn.jl" icon="flask" href="https://github.com/cstjean/ScikitLearn.jl">
    Julia implementation of the scikit-learn API, allowing use of Python's scikit-learn and Julia models interchangeably.
  </Card>
  <Card title="MLBase.jl" icon="toolbox" href="https://github.com/JuliaStats/MLBase.jl">
    A set of foundational functions to support the development of machine learning algorithms in Julia.
  </Card>
  <Card title="XGBoost.jl" icon="tree" href="https://github.com/dmlc/XGBoost.jl">
    eXtreme Gradient Boosting package in Julia, wrapping the highly optimized XGBoost C++ library.
  </Card>
</Columns>
