C and C++ remain the foundation of high-performance ML infrastructure. Many Python libraries (XGBoost, LightGBM, OpenCV) are C++ under the hood. This page covers the native libraries directly.

## General-purpose ML

<Columns cols={2}>
  <Card title="XGBoost" icon="bolt" href="https://github.com/dmlc/xgboost">
    Parallelized, optimized general-purpose gradient boosting library used extensively in competitions and production.
  </Card>
  <Card title="LightGBM" icon="feather" href="https://github.com/Microsoft/LightGBM">
    Microsoft's fast, distributed, high-performance gradient boosting framework based on decision tree algorithms.
  </Card>
  <Card title="CatBoost" icon="cat" href="https://github.com/catboost/catboost">
    Gradient boosting on decision trees with native categorical feature support and fast CPU/GPU inference.
  </Card>
  <Card title="mlpack" icon="cubes" href="https://www.mlpack.org/">
    Scalable C++ ML library with fast implementations of common algorithms: random forests, k-means, SVMs, neural networks.
  </Card>
  <Card title="Shogun" icon="toolbox" href="https://github.com/shogun-toolbox/shogun">
    The Shogun Machine Learning Toolbox — a large collection of ML algorithms with interfaces to Python, R, Java, Octave, and more.
  </Card>
  <Card title="DSSTNE" icon="server" href="https://github.com/amznlabs/amazon-dsstne">
    Amazon's software library for training and deploying deep neural networks with an emphasis on speed and scale on GPUs.
  </Card>
  <Card title="DLib" icon="face-smile" href="http://dlib.net/ml.html">
    Suite of ML tools for C++ including SVMs, deep learning, clustering, and image processing; easy to embed in other applications.
  </Card>
  <Card title="Vowpal Wabbit" icon="rabbit" href="https://github.com/VowpalWabbit/vowpal_wabbit">
    Fast out-of-core learning system supporting many ML tasks including contextual bandits and reinforcement learning.
  </Card>
</Columns>

## Deep learning

<Columns cols={2}>
  <Card title="Caffe" icon="coffee" href="https://github.com/BVLC/caffe">

  </Card>
  <Card title="CNTK" icon="microsoft" href="https://github.com/Microsoft/CNTK">

  <Card title="oneDNN" icon="circle-nodes" href="https://github.com/oneapi-src/oneDNN">
    Intel's open-source cross-platform performance library for deep learning primitives (convolutions, RNNs, batch norm).
  </Card>
  <Card title="DyNet" icon="arrows-rotate" href="https://github.com/clab/dynet">
    Dynamic neural network library for networks with dynamic structures; C++ core with Python bindings.
  </Card>
  <Card title="Speedster" icon="gauge-high" href="https://github.com/nebuly-ai/nebullvm/tree/main/apps/accelerate/speedster">
