Go and Rust have both developed meaningful ML ecosystems in recent years. Go's concurrency model makes it well suited for ML serving infrastructure, while Rust's memory safety and zero-cost abstractions are increasingly popular for high-performance inference. This page also covers a collection of smaller language ecosystems.
