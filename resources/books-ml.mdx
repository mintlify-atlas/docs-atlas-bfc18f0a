This page covers core machine learning textbooks, applied ML books, Bayesian methods, and the statistics and math foundations you'll need to go deep. All books below are either freely available online or link to the publisher page.

## Core ML textbooks

The gold-standard references used in university courses and research.

<Columns cols={2}>
  <Card title="The Elements of Statistical Learning" icon="book" href="https://web.stanford.edu/~hastie/ElemStatLearn/">
    Hastie, Tibshirani, and Friedman's comprehensive treatment of supervised and unsupervised learning methods. Free PDF from Stanford.
  </Card>
  <Card title="An Introduction to Statistical Learning (R)" icon="book" href="https://drive.usercontent.google.com/download?id=106d-rN7cXpyAkgrUqjcPONNCyO-rX7MQ&export=download">
    Accessible introduction to statistical learning with applications in R. Covers regression, classification, resampling, tree-based methods, and more.
  </Card>
  <Card title="An Introduction to Statistical Learning (Python)" icon="book" href="https://drive.usercontent.google.com/download?id=1ajFkHO6zjrdGNqhqW1jKBZdiNGh_8YQ1&export=download">
    The Python edition of the classic ISLR, with updated code examples.
  </Card>
  <Card title="Pattern Recognition and Machine Learning" icon="book" href="http://users.isr.ist.utl.pt/~wurmd/Livros/school/Bishop%20-%20Pattern%20Recognition%20And%20Machine%20Learning%20-%20Springer%20%202006.pdf">
    Bishop's seminal text on probabilistic graphical models, kernel methods, and Bayesian inference. A standard PhD-level reference.
  </Card>
  <Card title="Understanding Machine Learning" icon="book" href="http://www.cs.huji.ac.il/~shais/UnderstandingMachineLearning/">
    Shalev-Shwartz and Ben-David's theory-focused introduction. Covers PAC learning, VC dimension, and generalization bounds.
  </Card>
  <Card title="Foundations of Machine Learning" icon="book" href="https://cs.nyu.edu/~mohri/mlbook/">
    Mohri, Rostamizadeh, and Talwalkar cover the mathematical and theoretical foundations of ML algorithms.
  </Card>
</Columns>

### Additional core texts

- [The Hundred-Page Machine Learning Book](http://themlbook.com/wiki/doku.php) — A concise and practical overview of ML essentials.
- [A Course in Machine Learning](http://ciml.info/) — Hal Daumé III's accessible free textbook covering core algorithms.
- [Introduction to Machine Learning](https://alex.smola.org/drafts/thebook.pdf) — Alex Smola and S.V.N. Vishwanathan.
- [Introduction to Machine Learning](https://arxiv.org/pdf/0904.3664v1.pdf) — Amnon Shashua.
- [A First Encounter with Machine Learning](https://www.ics.uci.edu/~welling/teaching/273ASpring10/IntroMLBook.pdf)
- [Computer Age Statistical Inference (CASI)](https://web.stanford.edu/~hastie/CASI_files/PDF/casi.pdf) — Efron and Hastie.
- [Machine Learning, Neural and Statistical Classification](https://www1.maths.leeds.ac.uk/~charles/statlog/)
- [Mining Massive Datasets](http://infolab.stanford.edu/~ullman/mmds/book.pdf) — Stanford's large-scale data mining text.
- [Probabilistic Machine Learning](https://probml.github.io/pml-book/book1.html) — Kevin Murphy's comprehensive 2022 edition. Covers optimization, decision theory, information theory, linear models, and deep learning.
- [Probabilistic Machine Learning: Advanced Topics](https://probml.github.io/pml-book/book2.html) — Murphy's 2023 sequel covering diffusion models, generative modeling, and more advanced topics.
- [Mathematics for Machine Learning](https://mml-book.github.io/) — Covers the linear algebra, probability, and calculus used in ML.
- [Metalearning: Applications to Automated Machine Learning and Data Mining](https://link.springer.com/content/pdf/10.1007/978-3-030-67024-5.pdf) — Brazdil et al.

## Applied ML books

Hands-on books focused on practical implementation with popular frameworks and tools.

<Columns cols={2}>
  <Card title="Hands-On ML with Scikit-Learn and TensorFlow" icon="laptop-code" href="http://index-of.es/Varios-2/Hands%20on%20Machine%20Learning%20with%20Scikit%20Learn%20and%20Tensorflow.pdf">
    Aurélien Géron's practical guide to building ML systems with Python. Covers both classical algorithms and deep neural networks.
  </Card>
  <Card title="Data Mining: Practical ML Tools and Techniques" icon="database" href="https://cdn.preterhuman.net/texts/science_and_technology/artificial_intelligence/Data%20Mining%20Practical%20Machine%20Learning%20Tools%20and%20Techniques%202d%20ed%20-%20Morgan%20Kaufmann.pdf">
    Witten and Frank's hands-on guide to data mining with Weka. Thorough coverage of classification, clustering, and association rules.
  </Card>
  <Card title="Grokking Machine Learning" icon="book" href="https://www.manning.com/books/grokking-machine-learning">
    Introduces the most valuable machine learning techniques through intuitive explanations and worked examples.
  </Card>
  <Card title="Machine Learning Bookcamp" icon="laptop-code" href="https://www.manning.com/books/machine-learning-bookcamp">
    Alexey Grigorev's project-based approach to learning machine learning with Python.
  </Card>
</Columns>

### More applied texts

- [Real World Machine Learning](https://www.manning.com/books/real-world-machine-learning) — Free chapters available. Focuses on building production-ready ML systems.
- [Distributed Machine Learning Patterns](https://github.com/terrytangyuan/distributed-ml-patterns) — Free to read online. Patterns for scaling ML from a laptop to a distributed cluster.
- [Machine Learning with TensorFlow](https://www.manning.com/books/machine-learning-with-tensorflow)
- [Machine Learning Systems](https://www.manning.com/books/machine-learning-systems)
- [Graph-Powered Machine Learning](https://www.manning.com/books/graph-powered-machine-learning) — Alessandro Negro. Combining graph theory and models to improve ML projects.
- [Approaching Almost Any Machine Learning Problem](https://github.com/abhishekkrthakur/approachingalmost) — Abhishek Thakur. Free on GitHub.
- [MLOps Engineering at Scale](https://www.manning.com/books/mlops-engineering-at-scale) — Carl Osipov. Serverless ML in production.
- [Machine Learning Engineering in Action](https://www.manning.com/books/machine-learning-engineering-in-action) — Ben Wilson. Design patterns for deployable, maintainable ML projects.
- [Machine Learning System Design](https://www.manning.com/books/machine-learning-system-design) — Babushkin and Kravchenko.
- [Ensemble Methods for Machine Learning](https://www.manning.com/books/ensemble-methods-for-machine-learning) — Gautam Kunapuli.
- [Feature Engineering Bookcamp](https://www.manning.com/books/feature-engineering-bookcamp) — Maurucio Aniche.
- [Human-in-the-Loop Machine Learning](https://www.manning.com/books/human-in-the-loop-machine-learning) — Robert Monarch. Active learning, annotation, and transfer learning.
- [Automated Machine Learning in Action](https://www.manning.com/books/automated-machine-learning-in-action) — Song, Jin, and Hu. AutoKeras and Keras Tuner.
- [Privacy-Preserving Machine Learning](https://www.manning.com/books/privacy-preserving-machine-learning) — Chang, Zhuang, Samaraweera.
- [Machine Learning Algorithms in Depth](https://www.manning.com/books/machine-learning-algorithms-in-depth) — Vadim Smolyakov. Practical implementations of dozens of algorithms.
- [Practical Gradient Boosting](https://www.amazon.com/dp/B0BL1HRD6Z) — Guillaume Saupin.
- [Causal AI](https://www.manning.com/books/causal-machine-learning) — Robert Ness. Building AI models that reason about causality.
- [Bayesian Optimization in Action](https://www.manning.com/books/bayesian-optimization-in-action) — Quan Nguyen.
- [Optimization Algorithms](https://www.manning.com/books/optimization-algorithms) — Alaa Khamis.
- [AI-Powered Search](https://www.manning.com/books/ai-powered-search) — Grainger, Turnbull, Irwin.
- [Fight Fraud with Machine Learning](https://www.manning.com/books/fight-fraud-with-machine-learning) — Ashish Ranjan Jha.
- [Machine Learning for Drug Discovery](https://www.manning.com/books/machine-learning-for-drug-discovery) — Noah Flynn.
- [Managing Machine Learning Projects](https://www.manning.com/books/managing-machine-learning-projects) — Simon Thompson.
- [AI Governance](https://www.manning.com/books/ai-governance)
- [R Programming for Data Science](https://leanpub.com/rprogramming)
- [R for Data Science](https://r4ds.had.co.nz/) — Wickham and Grolemund.
- [Advanced R](http://adv-r.had.co.nz/) — Hadley Wickham.

## Statistics & math foundations

### Probability & Statistics

<Columns cols={2}>
  <Card title="Think Stats" icon="chart-bar" href="https://www.greenteapress.com/thinkstats/">
    Allen Downey's introduction to probability and statistics for programmers, using Python. Free online.
  </Card>
  <Card title="Introduction to Probability (MIT)" icon="chart-bar" href="http://athenasc.com/probbook.html">
    Bertsekas and Tsitsiklis. The textbook for MIT's probability course. Comprehensive and rigorous.
  </Card>
</Columns>

- [Introduction to Probability (Dartmouth)](https://math.dartmouth.edu/~prob/prob/prob.pdf)
- [Basic Probability Theory](https://www.math.uiuc.edu/~r-ash/BPT/BPT.pdf)
- [Probability & Statistics Cookbook](http://statistics.zone/) — A compact reference of formulas and definitions.
- [Probability Theory: The Logic of Science](https://bayes.wustl.edu/etj/prob/book.pdf) — E.T. Jaynes.
- [Introduction to statistical thought](https://people.math.umass.edu/~lavine/Book/book.pdf)
- [Introduction to Probability and Statistics Using R](http://ipsur.r-forge.r-project.org/book/download/IPSUR.pdf)
- [From Algorithms to Z-Scores](http://heather.cs.ucdavis.edu/probstatbook)
- [Practical Regression and Anova using R](https://cran.r-project.org/doc/contrib/Faraway-PRA.pdf) — Julian Faraway.
- [Forecasting: Principles and Practice](https://otexts.com/fpp2/) — Hyndman and Athanasopoulos. Free online.

### Linear Algebra

- [The Matrix Cookbook](https://www.math.uwaterloo.ca/~hwolkowi/matrixcookbook.pdf) — A compact reference for matrix identities and results.
- [Linear Algebra by Shilov](https://cosmathclub.files.wordpress.com/2014/10/georgi-shilov-linear-algebra4.pdf)
- [Linear Algebra Done Wrong](https://www.math.brown.edu/~treil/papers/LADW/LADW.html) — Treil. Free online.
- [Linear Algebra, Theory, and Applications](https://math.byu.edu/~klkuttle/Linearalgebra.pdf)
- [Convex Optimization](https://web.stanford.edu/~boyd/cvxbook/bv_cvxbook.pdf) — Boyd and Vandenberghe. Essential for understanding ML optimization.
- [Applied Numerical Computing](https://www.seas.ucla.edu/~vandenbe/ee133a.html) — Van den Berghe, UCLA.
- [The Math Behind Artificial Intelligence](https://www.freecodecamp.org/news/the-math-behind-artificial-intelligence-book)

### Calculus

- [Calculus Made Easy](https://github.com/lahorekid/Calculus/blob/master/Calculus%20Made%20Easy.pdf) — Silvanus Thompson's classic introduction.
- [Calculus by Ron Larson](https://www.pdfdrive.com/calculus-e183995561.html)
- [Active Calculus](https://scholarworks.gvsu.edu/books/20/) — Matt Boelkins. Free, inquiry-based text.
