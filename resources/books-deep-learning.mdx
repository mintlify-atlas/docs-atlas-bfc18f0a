Deep learning has its own rich body of literature, ranging from foundational theory to hands-on framework guides. This page covers the key books across deep learning fundamentals, framework-specific texts, NLP, and neural networks.

## Deep learning fundamentals

<Columns cols={2}>
  <Card title="Deep Learning (Goodfellow, Bengio, Courville)" icon="brain" href="https://www.deeplearningbook.org/">
    The definitive MIT Press textbook on deep learning. Covers mathematical foundations, feedforward networks, regularization, optimization, CNNs, RNNs, and generative models. Free online.
  </Card>
  <Card title="Deep Learning with Python" icon="brain" href="https://www.manning.com/books/deep-learning-with-python">
    François Chollet's practical introduction using Keras. Covers the core DL concepts through Python code, from basic networks to advanced architectures.
  </Card>
  <Card title="Deep Learning with Python, Second Edition" icon="brain" href="https://www.manning.com/books/deep-learning-with-python-second-edition">
    Updated edition of Chollet's book with new chapters on transformers, diffusion models, and modern best practices.
  </Card>
  <Card title="Deep Learning with Python, Third Edition" icon="brain" href="https://www.manning.com/books/deep-learning-with-python-third-edition">
    The latest edition, updated for the current state of deep learning.
  </Card>
</Columns>

### More foundational texts

- [Grokking Deep Learning](https://www.manning.com/books/grokking-deep-learning) — Andrew Trask. Builds intuition for deep learning from first principles without relying on libraries.
- [Inside Deep Learning](https://www.manning.com/books/inside-deep-learning) — Eric Heiden. Covers the math and code behind deep learning architectures.
- [Math and Architectures of Deep Learning](https://www.manning.com/books/math-and-architectures-of-deep-learning) — Covers the mathematical underpinnings of modern DL architectures.
- [Probabilistic Deep Learning with Python](https://www.manning.com/books/probabilistic-deep-learning-with-python) — Uncertainty estimation and probabilistic methods in deep learning.
- [Machine Learning for Business](https://www.manning.com/books/machine-learning-for-business) — Practical guide to applying DL and ML in business contexts.
- [AI Model Evaluation](https://www.manning.com/books/ai-model-evaluation) — Methods for evaluating the quality and behavior of AI models.

## Deep learning with specific frameworks

### JavaScript

- [Deep Learning with JavaScript](https://www.manning.com/books/deep-learning-with-javascript) — Neural networks in the browser and on Node.js using TensorFlow.js.

### R

- [Deep Learning with R, Third Edition](https://www.manning.com/books/deep-learning-with-r-third-edition) — Keras and TensorFlow from R. The third edition reflects the current Keras API.

### Structured data

- [Deep Learning with Structured Data](https://www.manning.com/books/deep-learning-with-structured-data) — Applying DL to tabular and relational data rather than images or text.

## Natural language processing

<Columns cols={2}>
  <Card title="Natural Language Processing in Action" icon="language" href="https://www.manning.com/books/natural-language-processing-in-action">
    Practical NLP with Python. Covers tokenization, embeddings, topic modeling, and sequence-to-sequence models.
  </Card>
  <Card title="Natural Language Processing in Action, Second Edition" icon="language" href="https://www.manning.com/books/natural-language-processing-in-action-second-edition">
    Updated edition covering transformers, BERT, and modern NLP pipelines.
  </Card>
</Columns>

- [Coursera Course Book on NLP](http://www.cs.columbia.edu/~mcollins/notes-spring2013.html) — Michael Collins's NLP lecture notes from Columbia. Free online.
- [NLTK Book](https://www.nltk.org/book/) — Natural Language Processing with Python using the NLTK library. Free online.
- [Foundations of Statistical Natural Language Processing](https://nlp.stanford.edu/fsnlp/promo/) — Manning and Schütze. A thorough statistical treatment of NLP.
- [Real-World Natural Language Processing](https://www.manning.com/books/real-world-natural-language-processing) — Building production NLP applications.
- [Essential Natural Language Processing](https://www.manning.com/books/essential-natural-language-processing) — Key concepts and techniques for NLP practitioners.
- [Deep Learning for Natural Language Processing](https://www.manning.com/books/deep-learning-for-natural-language-processing) — Applies deep learning methods specifically to NLP tasks.
- [Getting Started with Natural Language Processing](https://www.manning.com/books/getting-started-with-natural-language-processing) — Entry-level NLP guide.
- [Transfer Learning for Natural Language Processing](https://www.manning.com/books/transfer-learning-for-natural-language-processing) — Paul Azunre. Using pretrained models for downstream NLP tasks.

## Neural networks

<Columns cols={2}>
  <Card title="Neural Networks and Deep Learning" icon="network-wired" href="http://neuralnetworksanddeeplearning.com/">
    Michael Nielsen's free online book. Builds neural networks from scratch to build deep understanding. Highly recommended as a first read.
  </Card>
  <Card title="Graph Neural Networks in Action" icon="diagram-project" href="https://www.manning.com/books/graph-neural-networks-in-action">
    Covers graph neural network architectures and their application to relational and graph-structured data.
  </Card>
</Columns>

- [A Brief Introduction to Neural Networks](http://www.dkriesel.com/_media/science/neuronalenetze-en-zeta2-2col-dkrieselcom.pdf) — David Kriesel. A compact, free PDF introduction to neural network fundamentals.

## Reinforcement learning

- [Reinforcement Learning: An Introduction](http://incompleteideas.net/book/the-book-2nd.html) — Sutton and Barto. The standard textbook for RL. Free online, 2nd edition.
- [Reinforcement Learning](https://www.intechopen.com/books/reinforcement_learning) — Open access edited volume. Free.
