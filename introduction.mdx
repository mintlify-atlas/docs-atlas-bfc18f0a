Awesome Machine Learning is a community-curated reference catalog of machine learning frameworks, libraries, and software — organized by programming language. It was created to help researchers, engineers, and students quickly find the right ML tool for any task, no matter which language they work in.

<Note>
  This list is maintained by the community. If you want to add a library or fix an entry, see the [contributing guide](/contributing).
</Note>

## What's in the list

The list covers more than 30 programming languages, from Python and R to Go, Rust, Julia, Scala, and beyond. Within each language, entries are grouped by domain:

- **General-purpose machine learning** — scikit-learn, XGBoost, and similar toolkits
- **Deep learning** — PyTorch, TensorFlow, Keras, JAX
- **Natural language processing** — spaCy, Hugging Face Transformers, NLTK
- **Computer vision** — OpenCV, Detectron2, timm
- **Data analysis and visualization** — Pandas, NumPy, matplotlib, Plotly
- **Reinforcement learning, speech recognition, federated learning**, and more

<Tip>
  Python has the largest ML ecosystem in the list. Start with [Python: General-Purpose](/python/general-purpose) if you're unsure where to begin.
</Tip>

## How the list is organized

Every language section follows the same two-level structure: language first, then domain within that language.

```
Language (e.g., Python)
├── General-Purpose Machine Learning
├── Deep Learning
├── Natural Language Processing
├── Computer Vision
└── Data Analysis / Data Visualization
```

This means you can jump directly to the combination you need — for example, NLP in Julia or computer vision in C++.

## Deprecation policy

Entries in the list are marked **[Deprecated]** and retained for historical reference when either of the following is true:

- The repository owner explicitly states the library is no longer maintained.
- The repository has received no commits in **2–3 years**.

Deprecated entries are not removed so that existing projects that depend on them can still find documentation and context.

<Warning>
  If you're starting a new project, avoid libraries marked **[Deprecated]**. Check the linked repository for the last commit date before adopting any library.
</Warning>
